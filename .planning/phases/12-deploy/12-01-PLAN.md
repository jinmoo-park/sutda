**1. packages/server/src/index.ts 수정 — STATIC_DIR 경로를 import.meta.url 기반으로 변경**

서버 패키지가 `"type": "module"` + `module: "ESNext"` 빌드이므로 `__dirname`이 존재하지 않는다.
`import.meta.url` + `fileURLToPath`를 사용해야 한다.

파일 상단 import 영역에 추가:
```typescript
import { fileURLToPath } from 'url';
```

기존 20행의 STATIC_DIR 정의:
```typescript
const STATIC_DIR = join(process.cwd(), '../../packages/client/dist');
```

다음으로 변경:
```typescript
const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);
const STATIC_DIR = join(__dirname, '../../../packages/client/dist');
```

`dirname`은 이미 `import { join, extname } from 'path';`에서 import하고 있지 않으므로, path import를 수정:
```typescript
import { join, extname, dirname } from 'path';
```

경로 계산 검증:
- 빌드 후 `__dirname` = `{repo_root}/packages/server/dist/`
- `join(__dirname, '../../../packages/client/dist')` = `{repo_root}/packages/client/dist` (올바름)
