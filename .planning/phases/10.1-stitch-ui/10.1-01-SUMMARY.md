## Stitch MCP 도구 파라미터 실측 결과

이후 Wave에서 반드시 참조할 것:

### `create_project`
```json
{ "title": "프로젝트명" }
// 반환: { name: "projects/{id}", ... }
// projectId 추출: name.split('/')[1]
```

### `create_design_system`
```json
{
  "projectId": "...",
  "designSystem": {
    "displayName": "이름",
    "theme": {
      "colorMode": "DARK",
      "customColor": "#hex",
      "colorVariant": "TONAL_SPOT",
      "font": "폰트ID",
      "headlineFont": "폰트ID",
      "bodyFont": "폰트ID",
      "labelFont": "폰트ID",
      "designMd": "마크다운 가이드라인"
    }
  }
}
// 반환: { name: "assets/{id}", ... }
// designSystemId 추출: name.split('/')[1]
```

### `generate_screen_from_text`
```json
{
  "projectId": "...",
  "prompt": "컴포넌트 설명",
  "modelId": "GEMINI_3_FLASH"
}
```

### `apply_design_system`
```json
{
  "projectId": "...",
  "selectedScreenInstances": [],
  "assetId": "designSystemId"
}
```

### `list_screens` / `get_screen`
```json
// list_screens
{ "projectId": "..." }

// get_screen
{ "name": "projects/{projectId}/screens/{screenId}" }
// 반환에 HTML 포함
```
