    **Step 5: game-engine.ts 구현 (GREEN)**

    `packages/server/src/game-engine.ts` 생성:

    ```typescript
    import { createDeck } from '@sutda/shared';
    import type { Card, GameState, GameMode, PlayerState, RoomPlayer, GamePhase } from '@sutda/shared';
    ```

    핵심 구현:

    1. **constructor(roomId, players, mode, roundNumber=1)**
       - `createDeck()` 호출하여 deck 초기화 (셔플 안 된 상태)
       - players를 PlayerState로 매핑: `{ id, nickname, chips, cards: [], isAlive: true, isRevealed: false, currentBet: 0, isDealer: false, seatIndex }`
       - roundNumber=1이면 phase='dealer-select', 아니면 phase='attend-school'
       - pot=0, currentBetAmount=0, isTtong=false, attendedPlayerIds=[]

    2. **getCounterClockwiseOrder(dealerSeatIndex, totalPlayers)** (private):
       ```
       for i=1..totalPlayers: order.push((dealerSeatIndex - i + totalPlayers) % totalPlayers)
       return order  // 예: dealer=0, 4명 -> [3,2,1,0]
       ```

    3. **selectDealerCard(playerId, cardIndex)**:
       - phase가 'dealer-select'인지 검증
       - 이미 선택한 플레이어인지 검증
       - dealerSelectCards에 { playerId, cardIndex } 추가
       - 모든 플레이어 선택 완료 시:
         - `const kstHour = (new Date().getUTCHours() + 9) % 24`
         - isNight = kstHour >= 18 || kstHour < 6
         - isNight면 가장 낮은 rank, 아니면 가장 높은 rank 선택자가 dealer
         - 해당 플레이어의 isDealer=true, phase='attend-school'로 전환

    4. **setDealerFromPreviousWinner(winnerId)**: 해당 플레이어 isDealer=true

    5. **attendSchool(playerId)**:
       - phase가 'attend-school'인지 검증
       - 이미 등교했는지 검증
       - pot += 500, attendedPlayerIds에 추가
       - 모든 플레이어 등교 완료 또는 타임아웃 시: 미등교자 isAlive=false, phase='mode-select'

    6. **selectMode(playerId, mode)**:
       - dealer만 가능, phase='mode-select' 검증
       - state.mode = mode, phase='shuffling'

    7. **shuffle(playerId)**:
       - dealer만 가능, phase='shuffling' 검증
       - Fisher-Yates 셔플:
         ```
         const shuffled = [...state.deck];
         for (let i = shuffled.length - 1; i > 0; i--) {
           const j = Math.floor(Math.random() * (i + 1));
           [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
         }
         state.deck = shuffled;
         ```
       - phase='cutting'
       - cutterPlayerId 설정: dealer의 왼쪽 플레이어 = (dealerSeatIndex + 1) % totalPlayers

    8. **cut(playerId, cutPoints: number[], order: number[])**:
       - cutterPlayer만 가능, phase='cutting' 검증
       - cutPoints로 덱을 분할: cutPoints=[5,12]이면 [0..4], [5..11], [12..19] 3개 세그먼트
       - order=[2,0,1]이면 세그먼트를 [12..19, 0..4, 5..11] 순으로 재조립
       - 검증: 재조립 결과 20장, 원본과 카드 집합 동일
       - phase='dealing', dealCards() 자동 호출

    9. **declareTtong(playerId)**:
       - cutterPlayer만 가능, phase='cutting' 검증
       - isTtong=true, phase='dealing', dealCards() 자동 호출

    10. **dealCards()** (private):
        - 등교한 플레이어(isAlive===true)만 대상
        - dealerSeatIndex 찾기
        - order = getCounterClockwiseOrder(dealerSeatIndex, alivePlayers)
        - isTtong이면: order 순서대로 deck에서 2장씩 pop하여 cards에 push
        - 아니면: 1장씩 2라운드 (1라운드: order 순서대로 1장씩, 2라운드: 동일)
        - 배분 완료 후 phase='betting'
        - currentPlayerIndex = dealer의 seatIndex (선이 첫 베팅)

    11. **phase 유효성 검증**: 모든 public 메서드 진입 시 현재 phase를 체크하고 불일치 시 `throw new Error('INVALID_PHASE')` (per Claude's Discretion)
