  <action>
HandPanel.tsx의 카드 렌더링 영역(176행 부근 `<div className="flex gap-2 ...">`)을 수정한다.

1. 카드 개수가 3장인지 판별하는 변수 추가:
   ```
   const isThreeCards = cards.length === 3;
   ```

2. 카드 컨테이너의 className을 조건부로 변경:
   - 3장일 때: `flex items-center flex-wrap` (gap-2 제거)
   - 2장 이하: 기존 `flex gap-2 items-center flex-wrap` 유지

3. 각 카드 wrapper `<div key={origIdx}>` 에 3장일 때 겹치기 스타일 적용:
   - 첫 번째 카드(renderPos === 0): margin 없음
   - 두 번째 이후 카드: `ml-[-12px]` (negative margin-left 12px로 겹침)
   - z-index를 renderPos에 따라 증가: `z-[renderPos+1]` → style={{ zIndex: renderPos + 1 }}

구체적으로 카드 wrapper div에 다음을 추가:
```tsx
style={{
  ...getDealAnimStyle(renderPos),
  ...(isThreeCards && renderPos > 0 ? { marginLeft: '-12px' } : {}),
  zIndex: isThreeCards ? renderPos + 1 : undefined,
}}
```

기존 `style={getDealAnimStyle(renderPos)}`를 위의 합성 style로 교체한다.

주의: 2장 모드, 1장(한장공유) 모드에는 영향 없도록 `isThreeCards` 조건을 정확히 건다.
주의: 공개 카드(ring-2 ring-amber-400)가 겹침에 가려지지 않도록 z-index 처리. 공개 카드는 항상 renderPos 0(맨 앞)이므로 z-index 1로 자연스럽게 보인다.
