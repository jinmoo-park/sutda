## 변경 내용

### packages/server/src/index.ts

`leave-room` 핸들러 (`socket.on('leave-room', ...)`)에 다음 로직을 추가:

1. **이력 기록:** `engine.lastRoundHistory`가 있고 `gameHistories`에 아직 기록되지 않은 경우, push + `game-history` emit (중복 방지: `roundNumber` 기준)
2. **engine 플레이어 제거:** `engine.forcePlayerLeave(socket.id)` 호출 (try-catch로 안전 처리)
3. **engine chips → room chips 동기화:** `leaveRoom()` 호출 전에 engine의 정산된 chips를 room.players에 복사
4. **2인→1인 시 대기실 전환:** `leaveRoom()` 이후 활성 플레이어 < 2이면 `gamePhase='waiting'` + `gameEngines.delete()` + `room-state` emit
5. **3인 이상에서 1명 퇴장:** `room-state` broadcast; result phase였으면 `tryAdvanceNextRound()` 호출
6. **방 공실 정리:** `chatHistories`, `gameEngines`, `gameHistories` 모두 정리
