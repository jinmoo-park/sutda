```typescript
/** 족보 score 순서 배열 — 한 단계 차이 판별용 */
const SCORE_RANK_ORDER = [
  1300, 1200, 1100, // 광땡
  1010, 1009, 1008, 1007, 1006, 1005, 1004, 1003, 1002, 1001, // 땡
  60, 50, 40, 30, 20, 10, // 특수조합
  9, 8, 7, 6, 5, 4, 3, 2, 1, 0, // 끗
];

function isOneRankApart(scoreA: number, scoreB: number): boolean {
  const idxA = SCORE_RANK_ORDER.indexOf(scoreA);
  const idxB = SCORE_RANK_ORDER.indexOf(scoreB);
  if (idxA === -1 || idxB === -1) return false;
  return Math.abs(idxA - idxB) === 1;
}
```

2. **result phase SFX useEffect 수정:**
   현재 로직에서 승자/패자 구분 후 SFX를 결정하는 부분을 수정.

   **승자 분기 수정:**
   - 기존 승리 SFX 재생 후, 추가로 한 단계 차이 체크:

   - loserScore 계산: result phase에서 gameState.players 중 winnerId가 아닌 플레이어들의 cards로 evaluateHand 수행, 가장 높은 score를 가진 플레이어(2등)의 score 사용
   - 단, 다이한 플레이어(cards가 없거나 빈 배열)는 제외

   **패자 분기 수정:**

     try {
